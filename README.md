# Teave-frontend

## commit convention

| Type     | Description                   |
| -------- | ----------------------------- |
| feat     | Add a new feature             |
| fix      | Fix the bug                   |
| design   | UI design changes such as CSS |
| refactor | Refactoring the code          |
| docs     | Modify the document           |
| services | Change about server API       |
| test     | Make the test file            |
| chore    | etc.                          |
